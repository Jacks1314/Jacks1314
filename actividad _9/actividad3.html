El elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel de bloque. por ejemplo, puede crear un elemento <div> para que contenga todos los elementos del encabezado del sitio (el logotipo y la navegacion), o puede crear un elemento <div>para que contenga informacion en bloques

en un navegador, el contenido del elemento <div> comenzara en una nueva linea,pero aparte de estto no hara ninguna diferencia en la presentacion de la pagina. sin embargo, el uso de un atributo ID o Class en el espacio debe ocupar el elemento <div> significa que puede crear reglas de estilo CSS para indicar cuanto todos los elementos contenidos en el. tambien puede facilitar el seguimineto del codigo si ha utililizado elementos <div>para contener cada seccion de la pagina

Dado que puede haber varios otros elementos dentro de un elemento <div> puede ser util agregar un comentario despues de la etiqueta </div> de cierre. Esto le permite ver claramente a que etiqueta de apertura se supone que corresponde
