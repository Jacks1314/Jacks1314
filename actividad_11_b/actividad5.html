CSS Selectors

Hay muchos tipos diferentes de selector CSS que le permiten orientar las reglas a elementos especificados en un documento HTML.

Los selectores CSS distinguen entre mayusculas y minusculas, por lo que deben coincidir exactamente con los elementos y los valores de atributos.

Tipos de selectores
Selector Significado Ejemplo
Unicersal Selector Se aplica a todos los elemnetos de la pagina * {} Se dirige a todos los elementos de la pagina
Type selector Coincide con los hombres de elemento h1, h2, h3 {} corresponde a los elementos <h1>, <h2> y <h3>
Class Selector Coincide con un elemento cuyo atributo de clase tiene un valor que corresponde con el especificado despues del simbolo de punto "." . nota {} se dirige a cualquier elemento cuyo atributo de clase tenga un valor de nota
Id Selector Coincide con un elemento cuyo ID atributo tiene un valor que corresponde con el especificado despues del simbolo de hash "#" #introduction {} se dirige al elemento cuyo atributo ID tiene un valor de introduction
